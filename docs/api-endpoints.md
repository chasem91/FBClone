# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`

### Session

- `GET /api/session`
- `POST /api/session`
- `DELETE /api/session`

### Profiles

- `GET /api/profiles/:id`
- `POST /api/profiles`
- `PATCH /api/profiles`


### Friendships

- `GET /api/friendships`
- `POST /api/friendships`
- `GET /api/friendships/:id`
- `PATCH /api/friendships/:id`
- `DELETE /api/friendships/:id`

### Friends

- `GET /api/friends`

### Posts

- `GET /api/posts`
- `POST /api/posts/:id`
- `GET /api/posts/:id:`
- `PATCH /api/posts/:id:`
- `DELETE /api/posts/:id:`

### Newsfeed

- `GET /api/newsfeed`
